# discover.program.md

## Goal
- 바로 방문, 다운로드, 지원 가치가 있는 항목을 `discover`에 올린다.

## Success Conditions
- category 적합성
- action links 유효성
- summary 길이 안정성

## Failure Conditions
- broken CTA
- category mismatch
- stale source

## Allowed Surface
- `discover`
- `both`

## Escalation
- action link가 불안정하면 `exceptions`

## Discard
- 가치가 낮거나 이미 brief로 충분한 항목
