# brief.program.md

## Goal
- brief 초안은 한국어 해설 가치가 있는 source만 남긴다.

## Success Conditions
- source coverage가 충분하다.
- 제목, 요약, 본문이 한국어로 자연스럽다.
- 출처 링크가 모두 살아 있다.

## Failure Conditions
- source 부족
- quote boundary 불명확
- 중복 항목

## Allowed Surface
- `brief`
- `both`

## Escalation
- 인용 범위가 애매하면 `review`

## Discard
- 저가치 / 중복 / 정책 보류
